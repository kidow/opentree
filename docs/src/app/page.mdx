# Opentree

Free, open-source desktop app for building your link-in-bio page. No account required. Deploy to Vercel in one click.

## What is Opentree?

Opentree is a macOS desktop app that lets you create and publish a beautiful profile page — like Linktree, but free and self-hosted. Your data stays in a local JSON file you own.

## Features

- **Visual editor** — Drag-and-drop links, headings, and text blocks
- **Live preview** — Phone-shaped preview updates as you edit
- **Design control** — Customize accent, background, and surface colors
- **One-click deploy** — Publish to Vercel with your own API token
- **Local-first** — Config saved as `opentree.config.json` in your project folder
- **Open source** — MIT licensed, self-hostable, no vendor lock-in

## How it works

1. **Open a folder** — Choose any local folder as your project
2. **Edit** — Add links, headings, and text blocks in the Links tab
3. **Design** — Pick colors in the Design tab
4. **Publish** — Enter your Vercel token and deploy in the Publish tab

Your site is a single static `index.html` — no server, no database.
