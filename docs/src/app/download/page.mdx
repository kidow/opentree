# Download

Opentree is currently available for **macOS** (Apple Silicon and Intel).

## macOS

Download the latest `.dmg` from the [GitHub Releases](https://github.com/kidow/opentree/releases) page.

### System Requirements

- macOS 12 Monterey or later
- Apple Silicon (M1+) or Intel x86-64

### Install

1. Download `Opentree_x.x.x_aarch64.dmg` (Apple Silicon) or `Opentree_x.x.x_x64.dmg` (Intel)
2. Open the `.dmg` and drag **Opentree** to your Applications folder
3. On first launch, right-click → Open if macOS shows an unverified developer warning

## Build from source

```bash
git clone https://github.com/kidow/opentree
cd opentree/apps/desktop
npm install
source ~/.cargo/env
npx tauri build
```

Requires: Rust 1.75+, Node.js 20+, Xcode Command Line Tools.

## Windows / Linux

Not yet available as prebuilt binaries. You can build from source — Tauri supports all three platforms.
