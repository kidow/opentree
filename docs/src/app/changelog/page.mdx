# Changelog

## v0.1.0 — 2026-05-09

Initial release of the Opentree desktop app.

### Added

- macOS desktop app built with Tauri v2 + React
- Visual drag-and-drop editor for Link, Heading, and Text blocks
- Profile block with name, bio, and avatar URL
- Live phone-shaped preview
- Design tab: accent, background, and surface color pickers
- One-click Vercel deployment (REST API v13)
- Deploy state polling (BUILDING → READY)
- Local config saved as `opentree.config.json`
- Export to static `index.html` + `favicon.svg`
- Undo / Redo (50-step history)
- Import links from JSON file (append or replace)
- Unsaved-changes warning on window close (⌘W)
- Settings tab: site URL, project path, app version
- Keyboard shortcuts: ⌘S save, ⌘Z undo, ⌘⇧Z redo
