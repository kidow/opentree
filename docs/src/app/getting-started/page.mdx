# Getting Started

Have your profile page live on the internet in under 5 minutes.

## 1. Download and install

[Download Opentree for macOS](/download) and drag it to your Applications folder.

## 2. Open a project folder

On launch you'll see the Welcome screen. Click **"프로젝트 폴더 열기"** and pick any empty folder on your Mac — this will be your project folder.

Opentree creates `opentree.config.json` inside it.

## 3. Edit your profile

In the **Links** tab:

- Click the **Profile** block at the top to set your name, bio, and avatar URL
- Click **"+ 블록 추가"** to add Link, Heading, or Text blocks
- Drag blocks to reorder them
- Click any block to edit its content inline

The phone preview on the right updates live.

## 4. Customize design

Switch to the **Design** tab to choose:

- **Accent color** — buttons and link highlights
- **Background color** — page background
- **Surface color** — card backgrounds

## 5. Save

Press **⌘S** or click the **저장** button in the sidebar. This writes `opentree.config.json` to your project folder.

## 6. Publish to Vercel

1. Go to the **Publish** tab
2. Get a token from [vercel.com/account/tokens](https://vercel.com/account/tokens)
3. Paste the token and click **저장**
4. Enter a project name (lowercase, hyphens only)
5. Click **"Vercel에 배포"**

Your site will be live at `https://<project-name>.vercel.app` within seconds.

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| ⌘S | Save |
| ⌘Z | Undo |
| ⌘⇧Z | Redo |

## Export (optional)

Use **내보내기** in the sidebar to save `index.html` and `favicon.svg` locally — deploy them anywhere that hosts static files.
