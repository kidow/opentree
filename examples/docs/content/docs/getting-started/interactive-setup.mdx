# Interactive setup

## What this page solves

You want a starter config, but you would rather answer prompts than pass flags to `init`.

## Run the setup

```bash title="Terminal"
opentree interactive
```

The prompt flow collects:

1. name
2. bio
3. avatar URL
4. site URL
5. template
6. first link title
7. first link URL

## Important behavior

- the command writes `opentree.config.json` only after validating the generated config
- if the config file already exists, interactive setup exits with an error instead of overwriting it

## When to use this

Use `interactive` for the first pass. After that, use focused commands such as `profile set`, `theme set`, and `link add`.
