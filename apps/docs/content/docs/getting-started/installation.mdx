# Installation

## What this page solves

You want the CLI installed globally and you want to confirm which package name and command name matter.

## Requirements

- Node.js `>=22.12.0`
- npm available on your machine

## Install the package

```bash title="Terminal"
npm install -g opentree-cli
```

The published package name is `opentree-cli`. The installed command is `opentree`.

## Confirm the command is available

```bash title="Terminal"
opentree --help
opentree --version
```

Expected result:

- `opentree --help` prints the task-oriented command list from `src/cli.js`
- `opentree --version` prints the package version from the root `package.json`

## Common failure

If `opentree` is not found after install, your global npm bin directory is probably not on `PATH`.
