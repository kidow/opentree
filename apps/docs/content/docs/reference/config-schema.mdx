# Config schema

| Field | Type | Constraint |
| --- | --- | --- |
| `schemaVersion` | integer | optional, but when present it must be `1` |
| `profile.name` | string | required, non-empty |
| `profile.bio` | string | required |
| `profile.avatarUrl` | string | required, empty string or `http`/`https` URL |
| `links` | array | required, at least one item |
| `links[].title` | string | required, non-empty |
| `links[].url` | string | required, `http`/`https` URL |
| `theme.accentColor` | string | required, `#RRGGBB` |
| `theme.backgroundColor` | string | required, `#RRGGBB` |
| `theme.textColor` | string | required, `#RRGGBB` |
| `template` | string | optional, `glass` or `terminal` |
| `siteUrl` | string | optional, empty string or `http`/`https` URL |
| `analytics.clickTracking` | string | optional, `off` or `local` |
| `metadata.title` | string | optional |
| `metadata.description` | string | optional |
| `metadata.ogImageUrl` | string | optional, empty string or `http`/`https` URL |
| `metadata.socialCard.eyebrow` | string | optional |
| `metadata.socialCard.style` | string | optional, `glass` or `terminal` |
| `metadata.socialCard.showQrCode` | boolean | optional |

## Source of truth

- runtime validation: `src/config.js`
- editor schema: `opentree.schema.json`
- starter defaults: `src/init.js`

## Recommendation

Use the CLI editing commands for day-to-day changes and keep manual JSON edits for bulk review or automation.
