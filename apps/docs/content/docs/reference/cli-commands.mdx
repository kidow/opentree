# CLI commands

## Bootstrapping

- `opentree init`
- `opentree interactive`
- `opentree validate`

## Build and deploy

- `opentree build`
- `opentree dev`
- `opentree deploy`
- `opentree doctor`

## Config editing

- `opentree config show`
- `opentree profile set`
- `opentree site set`
- `opentree meta set`
- `opentree theme set`

## Link management

- `opentree link add`
- `opentree link list`
- `opentree link move`
- `opentree link preset`
- `opentree link remove`
- `opentree link update`

## Import and deterministic shortcuts

- `opentree import links`
- `opentree prompt`

## Vercel integration

- `opentree vercel link`
- `opentree vercel status`
- `opentree vercel unlink`

## Shell support

- `opentree completion bash`
- `opentree completion zsh`
