# CLI commands

## Bootstrapping

- `opentree init`
- `opentree interactive`
- `opentree validate`

## Build and deploy

- `opentree build`
- `opentree dev`
- `opentree deploy`
- `opentree doctor`
- `opentree schema [command] --json`

## Config editing

- `opentree config show`
- `opentree profile set`
- `opentree site set`
- `opentree meta set`
- `opentree theme set`

## Link management

- `opentree link add`
- `opentree link list`
- `opentree link move`
- `opentree link preset`
- `opentree link remove`
- `opentree link update`

## Import and deterministic shortcuts

- `opentree import links`
- `opentree prompt`

## Vercel integration

- `opentree vercel link`
- `opentree vercel status`
- `opentree vercel unlink`

## Shell support

- `opentree completion bash`
- `opentree completion zsh`

## Agent-first helpers

- use `--json` on operational commands when you need a stable machine-readable contract
- use `--dry-run` on mutating commands to preview changes without writing files or invoking remote deploy steps
- use `opentree schema --json` or `opentree schema <command> --json` to inspect supported flags, required values, and enums at runtime

## Accuracy note

The CLI help text in `src/cli.js` is the source of truth for this command map.
