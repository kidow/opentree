# Preview and build

## What this page solves

You want to preview changes locally and know exactly which files are emitted by the build step.

## Local preview

```bash title="Terminal"
opentree dev
opentree dev --port 4321
```

Behavior:

- the server validates `opentree.config.json` before it starts
- the preview re-renders on refresh using the current config
- when the config is invalid, the browser shows a problem page instead of stale content

## Production bundle

```bash title="Terminal"
opentree build
```

Default output:

- `dist/index.html`
- `dist/favicon.svg`
- `dist/opengraph-image.svg`

Conditional output when `siteUrl` exists:

- `dist/robots.txt`
- `dist/sitemap.xml`

## Extra behavior worth knowing

- `metadata.ogImageUrl` can override the generated OG image
- `metadata.socialCard.showQrCode` renders a QR block only when `siteUrl` is configured
- `analytics.clickTracking: "local"` stores click counts in `localStorage`
