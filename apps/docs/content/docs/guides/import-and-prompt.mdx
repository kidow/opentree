# Import and prompt

## What this page solves

You want to batch-add links or make fast deterministic edits with limited natural-language support.

## Import links from JSON

```bash title="Terminal"
opentree import links --file ./links.json
opentree import links --file ./links.json --replace
```

Accepted input formats:

- a JSON array of `{ "title": "...", "url": "..." }`
- an object with a `links` array using the same shape

## Apply supported prompt shortcuts

```bash title="Terminal"
opentree prompt "set my bio to Shipping links"
opentree prompt "set template to terminal"
opentree prompt "add link Docs https://example.com/docs"
opentree prompt "add github kidow"
```

## Important accuracy note

`prompt` is not an AI editor. It uses deterministic regex matches from `src/prompt.js`, so unsupported phrasing does nothing.
