# Editing profile

## What this page solves

You want to change public-facing identity and metadata through targeted commands.

## Profile fields

```bash title="Terminal"
opentree profile set --name "Kidow" --bio "Shipping links" --avatar-url "https://example.com/avatar.png"
```

## Site URL and analytics

```bash title="Terminal"
opentree site set --url "https://links.example.com" --analytics local
```

## Metadata and social card settings

```bash title="Terminal"
opentree meta set --title "Kidow Links" --description "Operator manual"
opentree meta set --card-eyebrow "Operator Manual" --card-style terminal --show-qr-code
```

## Theme colors

```bash title="Terminal"
opentree theme set --accent-color "#0f766e" --background-color "#ecfeff" --text-color "#042f2e"
```

## What changes

These commands update `opentree.config.json`, validate the result, and reject invalid combinations before saving.
